📚 BookVerse – Digital Library Application

BookVerse is a responsive digital library web application built using React. It allows users to search, explore, and manage books with real-time data fetched from the Google Books API.

🚀 Features 🔍 Search books by title, author, or keywords 📂 Filter books by categories 📖 View detailed information (author, publish date, description) 🔗 "Read Now" feature to access books via external links ❤️ Save books to Favorites using localStorage ⚡ Fast and dynamic UI using React components

🛠️ Tech Stack Frontend: React, Tailwind CSS API: Google Books API Concepts Used: Asynchronous JavaScript (fetch, async/await) State & Props (React) Component-based architecture Local Storage (data persistence)

🌐 API Used

This project uses the Google Books API to fetch real-time book data.
